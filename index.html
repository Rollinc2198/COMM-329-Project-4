Maze Bot

Maze Bot

A Python2/Pygame experiment inspired by XScreenSaver's automatic maze generating/solving screensaver. Once started, the program will attempt to solve the maze given using Dijkstra's algorithm to find the shortest path through the maze. It starts out knowing nothing. As it explores, it marks paths seen and paths explored.

3D Prism

3D Prism

A 3D prism created in WebGL. The prism has a different color on each vertice. The prism can be moved using w, a, s, and d. It can rotate in the x, y, and z plance by hitting x, y, and z, respectively. It also can be scaled in the x and z plane by hitting the right arrow key and the down arrow key, respectively.

Go to 3D Prism

2D Maze

2D Maze

A 2D maze built in WebGL. The blue square can be moved using w, a, s, and d. The goal is to reach the red square without running into any walls. If the player runs into a wall, they are set back to the beginning.
